Etiquetas en HTML

Etiqueta ¿Para qué sirve? Ejemplo
meta / <meta> / Define los metadatos en la página "Estos son algunos ejemplos de metadatos":
<meta charset="UTF-8"> -> Charset especifica el conjunto de caracteres
<meta name="viewport" content="width=device-width, initial-scale=1.0"> -> Viewport ayuda la relación de aspecto de la página entre dispositivos, especialmente en celulares
title / <title> / Le da el titulo a la página, este puede ser visible en el navegador al momento de ver la pestaña<title><Apunte de etiquetas en HTML></title>
heading / <hn> / Define hasta seis encabezados, con cada encabezado que se coloque, el tamaño de letra va disminuyendo

Encabezado 1

Encabezado 2

Encabezado 3

párrafo / <p> / Sirve para escribir un párrafo en la página hasta seis encabezados <p><Esto es un párrafo corto></p>
Unordered List (lista desordenada) / <ul> / Crea listas sin un orden en específico 
 
<ul> 
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>
