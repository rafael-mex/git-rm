Explicando un código de C en HTML

Código de C:


#include <stdio.h>

int main() {
    int edad;
    printf("¿Cuántos años tienes? ");
    scanf("%d", &edad);

    if (edad >= 18) {
        printf("Eres mayor de edad.\n");
    } else {
        printf("Eres menor de edad.\n");
    }

    return 0;
}

Explicación del código:

Descripción:
El código del programa en C le solicita al usuario su edad y luego verifica si es mayor o menor. De esta manera: 

#include <stdio.h> -> Incluye la biblioteca estándar de entrada y salida, necesaria para usar funciones como printf y scanf

int main() { -> Es la función principal, desde aquí comienza la ejecución del código, se abren llaves para indicar el incio y final del código. 
    

    int edad; -> Declara que la variable "edad" es de tipo entero.
    

    printf("¿Cuántos años tienes? "); -> Imprime el mensaje de "¿Cuántos años tienes?" en la consola.
    

    scanf("%d", &edad); -> 
    scanf: Escanea la entrada del usuario, en este caso, un número. 
    %d: Es un especificador de formato, sirve para leer números enteros o decimales. 
    &edad: Indica que el valor ingresado por el usuario se guardará en lavariable de "edad", anteriormente declarada como entero.
    


    if (edad >= 18) { -> Condicional if: Revisa si el número almacenado en "edad" es mayor o igual a 18, si se cumple esta condición, se ejecuta el código contenido dentro de las llaves. 
    

        printf("Eres mayor de edad.\n"); -> Imprime el mensaje "Eres mayor de edad" en la consola. 
 \n : Significa salto de línea.
    

    } else { -> Si la condicion de "if" no se cumple, entonces se ejecuta el código contenido dentro de las llaves, que inicia después del "else". 
    

        printf("Eres menor de edad.\n"); -> Imprime el mensaje "Eres menor de edad" en la consola.
 \n : Significa salto de línea.
    }
    

    return 0; -> Indicca que el programa ha finalizado sin errores. 
}
